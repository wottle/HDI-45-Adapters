# HDI-45 to HD-15 (VGA) Adapter

This version of the adapter will provide a female HD-15 connector to allow you to plug in a VGA monitor (as long as it supports 640x480@67Hz or 832x624@75Hz.  

The gerbers were included in the "gerber" directory as a zip file, and the case is included as two parts (bottom and top) in the "case" directory
